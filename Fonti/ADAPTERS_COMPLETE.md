## 1. Product Overview

### Introduction

**TK3/TK4 Mounting Adapters** provide flexible installation solutions for Teklab oil level regulators across a wide range of compressor brands and models. The adapter system enables **right or left mounting configurations** to accommodate different installation requirements, piping layouts, and space constraints in refrigeration systems.

With **10 adapter types** covering thread standards (UNEF, NPT, GAS, Metric), Rotalock connections, flange mounting, and direct injection applications, the adapter family ensures compatibility with virtually all commercial and industrial compressors from major manufacturers including Bitzer, Bock, Copeland, Dorin, Frascold, Carrier, and others.

### Key Positioning

| Feature | Adapter Advantage |
|---------|-------------------|
| **Universal Compatibility** | Covers Bitzer, Bock, Copeland, Dorin, Frascold + 5 more brands |
| **Right/Left Flexibility** | Same adapter, rotate for desired orientation |
| **10 Adapter Types** | Thread (UNEF/NPT/GAS/Metric), Rotalock (2 sizes), Flange (2 heights), Direct Injection |
| **Direct Mounting Option** | Many compressors require no adapter (Code 0) |
| **Flange as Spacer** | Codes 3 and 7 double as 50mm/22mm spacers |

### Primary Applications

**Compressor Crankcase Mounting:**
- **Reciprocating Compressors**: Bitzer, Bock, Copeland, Dorin, Frascold semi-hermetic models
- **Scroll Compressors**: Copeland ZF/ZS, Bitzer ZL/ZM with NPT adapter
- **Screw Compressors**: Bitzer HSK/HSN, Bock HGX with flange or UNEF adapter

**Oil Separator Mounting:**
- Flange adapters (Code 3 or 7) for separator vessel mounting
- Rotalock adapters (Code 5 or 8) for Rotalock-style oil separators

**Custom Mounting:**
- Liquid receivers, oil reservoirs, or custom vessels requiring level regulation
- Retrofit applications where original equipment uses non-standard mounting

### Why Use Adapters?

**When Adapter Required:**
- ✅ Compressor mounting port thread/flange differs from TK3/TK4 direct mount provision
- ✅ Right or left mounting orientation needed for piping layout optimization
- ✅ Mounting height adjustment required (use flange spacers Code 3 or 7)
- ✅ Rotalock connection standard in system (Code 5 or 8)

**When No Adapter Needed (Code 0):**
- ✅ Compressor has direct mounting provision compatible with TK3/TK4 base thread
- ✅ Most compact installation desired (no additional height)
- ✅ Cost optimization (eliminate adapter component)

## 2. Adapter Types & Features

### Standard Thread Adapters

#### Code 1: 1-1/8" - 18 UNEF (TKX-A001)
- **Thread**: UNEF (Unified Extra Fine Thread) 1-1/8" - 18 TPI
- **L1 Dimension**: 34 mm additional height
- **Common Applications**: 
  - Bitzer 2-series compressors (2DC, 2EC, 2FC, 2GC, 4CC, 4DC)
  - Bock HA12/22/34, HG12/22/34, HGX12P/22P series
  - Dorin K100/150/180/200/400 models, H1, SCC250-380
  - L'Unité Hermétique TAG/TAH, Maneurope LT/MT/SM/SZ
- **Key Feature**: Standard HVAC&R thread, widely used in medium-displacement compressors

#### Code 2: ¾" NPT (TKX-A002)
- **Thread**: NPT (National Pipe Thread) ¾" tapered
- **L1 Dimension**: 34 mm additional height
- **Common Applications**:
  - Bitzer scroll compressors (ZL, ZM series)
  - Copeland scroll compressors (ZF, ZS series)
- **Key Feature**: North American standard thread, common in scroll compressor oil management

#### Code 4: 1" GAS (TKX-A004)
- **Thread**: GAS (BSP) 1" cylindrical or tapered
- **L1 Dimension**: 43 mm additional height
- **Common Applications**: European equipment with BSP thread standard
- **Key Feature**: British Standard Pipe thread (ISO 228-1)

#### Code 6: M36 × 1.5 (TKX-A006)
- **Thread**: Metric M36 × 1.5 mm pitch
- **L1 Dimension**: 45 mm additional height
- **Common Applications**: European compressors with metric thread mounting
- **Key Feature**: Fine-pitch metric thread for precision installation

### Rotalock Adapters

#### Code 5: Rotalock 1" ¾ - 12 UNF (TKX-A005)
- **Connection**: Rotalock 1.75" diameter (1" ¾ size), 12 UNF thread
- **L1 Dimension**: 53.4 mm additional height
- **Common Applications**: Smaller Rotalock ports on compressors, oil separators
- **Installation Torque**: **50 N·m** (requires Rotalock service tool)
- **Key Feature**: Service valve capability, allows TK3/TK4 removal without system depressurization

#### Code 8: Rotalock 1" ¼ - 12 UNF (TKX-A008)
- **Connection**: Rotalock 2.25" diameter (2" ¼ size), 12 UNF thread
- **L1 Dimension**: 41.4 mm additional height
- **Common Applications**: Larger Rotalock ports on compressors, oil separators
- **Installation Torque**: **50 N·m** (requires Rotalock service tool)
- **Key Feature**: Higher flow capacity than Code 5 for larger systems

### Flange Adapters (Also Function as Spacers)

#### Code 3: 3/4/6 Holes Flange - 50mm (TKX-A003)
- **Bolt Pattern**: Accommodates 3-hole, 4-hole, or 6-hole flange configurations
- **L1 Dimension**: 50 mm spacer height
- **Common Applications**:
  - Bitzer 4G/4H/4J/4NC/4PC/4TC/4VC/6F/6G/6H/8FC/S4/S6 series
  - Bock HA4/5/6, HG4/5/6/7/8, HGX4 series
  - Copeland 4CC/6CC, D-series (D2/D3/D4/D6/D9/DM)
  - Dorin K-series (except Code 1 models), H-series, SCC500-2500
  - Frascold A/B/D/F/Q/S/V/W/Z series
  - Carrier EA/ER, Prestcold P-series, Arctic Circle G-series
- **Key Feature**: Dual function - flange adapter OR 50mm spacer for clearance adjustment
- **Included Hardware**: Bolts, nuts, gasket (if applicable)

#### Code 7: 3/4/6 Holes Flange - 22mm (TKX-A0070)
- **Bolt Pattern**: Accommodates 3-hole, 4-hole, or 6-hole flange configurations
- **L1 Dimension**: 22 mm spacer height
- **Common Applications**: Same compressor families as Code 3, when shorter spacer preferred
- **Key Feature**: Lower profile than Code 3 (22mm vs 50mm) for space-constrained installations
- **Included Hardware**: Bolts, nuts, gasket (if applicable)

### Direct Injection Adapters

#### Code 9: Direct Injection - 26 bar MOPD (TKX-ADI2)
- **Application**: Direct refrigerant injection for 26 bar MOPD (Maximum Operating Pressure Differential) models
- **L1 Dimension**: Varies by application (consult factory)
- **Key Feature**: Specialized adapter for direct injection oil return systems

#### Code 10: Direct Injection - Other Models (TKX-ADI3)
- **Application**: Direct refrigerant injection for models other than 26 bar MOPD
- **L1 Dimension**: Varies by application (consult factory)
- **Key Feature**: Custom design for specific direct injection requirements

### Code 0: No Adapter (Direct Mounting)

- **Description**: TK3/TK4 mounts directly to compressor without intermediate adapter
- **L1 Dimension**: 0 mm (no additional height)
- **Common Applications**: Compressors with direct mounting provision matching TK3/TK4 base thread
- **Advantages**: Most compact installation, lowest cost, fewest leak points
- **Selection**: Available for many compressor models (see Selection Table - often listed as "0 or 3")

### Right / Left Mounting Capability

**Universal Design**: All adapters support **both right and left mounting** orientations.

**How It Works**:
- **Right Mounting**: Oil level regulator body projects to right side of compressor
- **Left Mounting**: Oil level regulator body projects to left side of compressor
- **Implementation**: Rotate TK3/TK4 unit on adapter or reposition adapter installation to achieve desired orientation

**No Special Ordering**: Same adapter part number works for right or left - specify orientation during installation, not during order.

### Selection Code Interpretation

#### Code "0 or 3" - Two Options Available

**Option 1: Code 0 (Direct Mounting)**
- **No adapter required** - TK3/TK4 mounts directly to compressor
- **Advantages**: Most compact (no L1 height), lowest cost, fewest leak points
- **When to Use**: Compressor has direct mounting provision, space is constrained, simplicity preferred

**Option 2: Code 3 (Flange 50mm - TKX-A003)**
- **Flange adapter** with 50mm spacer height (L1 = 50mm)
- **Advantages**: Flexible mounting, can function as spacer, accommodates 3/4/6-hole patterns
- **When to Use**: Flange mounting required, need 50mm clearance adjustment, or prefer standardized flange interface

**How to Choose**: Verify compressor documentation for mounting provisions. If uncertain, choose Code 3 (flange) for maximum flexibility.

#### Code "1" - UNEF Adapter Required

**TKX-A001** (1-1/8" - 18 UNEF thread adapter)
- **Common in**: Medium-displacement compressors (Bitzer 2/4-series, Bock HA/HG12-34, Dorin K100-400, L'Unité, Maneurope)
- **L1**: 34mm additional height
- **Standard thread**: UNEF widely used in HVAC&R industry

#### Code "2" - NPT Adapter Required

**TKX-A002** (¾" NPT thread adapter)
- **Common in**: Scroll compressors (Bitzer ZL/ZM, Copeland ZF/ZS)
- **L1**: 34mm additional height
- **Standard thread**: NPT (North American tapered pipe thread)

## 5. Installation Guidelines

### Pre-Installation Planning

**Step 1: Verify Adapter Selection**
- Confirm compressor brand/model matches Selection Table
- Verify adapter ordering code (TKX-Axxx) received matches selection
- Check adapter condition (no thread damage, flange flatness, O-ring integrity for Rotalock)

**Step 2: Calculate Total Height & Clearances**
- Total height = Base TK3/TK4 height + L1 (adapter)
- Top clearance required = 150-200mm + L1
- Radial clearance = 60-100mm (depending on adapter type)
- **Verify adequate space** before starting installation

**Step 3: Determine Right or Left Mounting**
- Based on piping layout, maintenance access, space constraints
- Same adapter works for both - plan orientation before installation

**Step 4: Gather Tools & Materials**
- Calibrated torque wrench (appropriate range for adapter type)
- Rotalock service tool (if Code 5 or 8)
- Thread sealant: PTFE tape or anaerobic paste (refrigeration-compatible)
- Hex wrenches or socket set (for flange bolts if Code 3/7)
- Installation manual (TK3+/TK4 and adapter-specific)

### Installation Procedures by Adapter Type

#### Thread Adapters (Codes 1, 2, 4, 6)

**Procedure**:

1. **Clean Threads**:
   - Remove metal chips, old sealant from compressor mounting port
   - Inspect threads for damage (burrs, cross-threading, corrosion)

2. **Apply Sealant**:
   - Wrap PTFE tape 2-3 turns on adapter male thread (wrap in direction of tightening)
   - **OR** apply thin layer of anaerobic thread sealant (per manufacturer instructions)
   - **Critical**: Keep sealant away from first 2 threads (prevent contamination into system)

3. **Thread Adapter Hand-Tight**:
   - Thread adapter into compressor port by hand
   - Ensure smooth threading (no resistance = good thread engagement)
   - Stop when hand-tight (adapter snug but not torqued)

4. **Orient for Right/Left Mounting**:
   - Rotate adapter to desired orientation (consider piping layout, TK3/TK4 body clearance)
   - **Note**: Final torque may rotate adapter slightly - plan for ~30-45° rotation during torquing

5. **Torque to Specification**:
   - Use calibrated torque wrench with appropriate socket
   - **Code 1 (UNEF)**: 25-35 N·m
   - **Code 2 (NPT)**: 30-40 N·m (tapered thread - do NOT overtighten)
   - **Code 4 (GAS)**: 30-40 N·m
   - **Code 6 (M36)**: 35-45 N·m (fine pitch - exact torque important)

6. **Verify Adapter Orientation**:
   - Check final adapter position allows TK3/TK4 body clearance
   - If orientation incorrect, remove adapter, reapply sealant, reinstall with adjusted starting position

7. **Install TK3/TK4 Unit**:
   - Thread TK3/TK4 onto adapter (hand-tight, then wrench-tighten per TK3/TK4 manual)
   - Verify oil level regulator body in desired right/left orientation

#### Rotalock Adapters (Codes 5, 8)

**Procedure**:

1. **Inspect Rotalock Port & Adapter**:
   - Verify Rotalock port on compressor/vessel is clean (no debris, old O-ring fragments)
   - Check adapter Rotalock seal (O-ring) for damage (replace if cut, cracked, or hardened)
   - Verify adapter thread size matches port (Code 5 = 1" ¾, Code 8 = 2" ¼)

2. **Lubricate O-Ring**:
   - Apply thin film of refrigeration oil (POE, mineral, or PAG - match system oil) to adapter O-ring
   - **Purpose**: Ease installation, prevent O-ring damage during tightening

3. **Position Adapter on Rotalock Port**:
   - Align adapter with Rotalock port (O-ring seated in groove)
   - Thread Rotalock nut onto port threads hand-tight

4. **Torque with Rotalock Service Tool**:
   - Use **Rotalock service tool** (special wrench for Rotalock nuts)
   - **Critical Torque**: **50 N·m** for both Code 5 and Code 8
   - **Warning**: Under-torquing causes leaks; over-torquing damages O-ring or threads

5. **Verify Seal Integrity**:
   - Pressurize system (slowly, if possible) and leak-check adapter connection with soap solution or electronic leak detector
   - **If leak detected**: Depressurize, remove adapter, inspect O-ring, replace if damaged, retorque to 50 N·m

6. **Install TK3/TK4 Unit**:
   - Thread TK3/TK4 onto adapter top connection
   - Orient regulator body for right/left mounting as desired

#### Flange Adapters (Codes 3, 7)

**Procedure**:

1. **Verify Flange Compatibility**:
   - Check compressor flange bolt pattern (3-hole, 4-hole, or 6-hole)
   - Confirm adapter TKX-A003 or TKX-A0070 accommodates pattern
   - **Code 3**: 50mm height; **Code 7**: 22mm height

2. **Prepare Flange Surfaces**:
   - Clean compressor flange surface (remove old gasket material, oil residue)
   - Inspect flange flatness (no warping, deep scratches)
   - Clean adapter flange surface

3. **Install Gasket** (if applicable):
   - Place gasket on compressor flange (centered on bolt holes)
   - **OR** use liquid gasket sealant per manufacturer instructions (some applications)

4. **Position Adapter on Flange**:
   - Align adapter flange with compressor flange (match bolt holes)
   - Insert bolts through adapter and compressor flange holes (hand-thread nuts)

5. **Tighten Bolts in Pattern**:
   - **3-hole**: Alternate pattern (e.g., 1-3-2)
   - **4-hole**: Star pattern (e.g., 1-3-2-4, opposite bolts alternating)
   - **6-hole**: Star pattern (e.g., 1-4-2-5-3-6)
   - **First Pass**: Snug all bolts hand-tight
   - **Second Pass**: Torque to 50% of final spec (even loading)
   - **Third Pass**: Torque to 100% of final spec per compressor manufacturer (typically 15-25 N·m for M8-M10 bolts)

6. **Verify Flange Seal**:
   - Inspect for even gasket compression (no gaps visible)
   - Pressurize system and leak-check flange joint

7. **Install TK3/TK4 Unit**:
   - Thread TK3/TK4 onto adapter top connection
   - Orient regulator body for right/left mounting as desired

#### Direct Mounting (Code 0)

**Procedure**:

1. **Verify Direct Mount Compatibility**:
   - Confirm compressor has direct mounting provision matching TK3/TK4 base thread
   - Check mounting port condition (clean threads, no damage)

2. **Apply Thread Sealant**:
   - PTFE tape or anaerobic sealant on TK3/TK4 base thread (same as thread adapter installation)

3. **Thread TK3/TK4 Directly onto Compressor**:
   - Hand-tighten TK3/TK4 into compressor mounting port
   - Torque per TK3/TK4 installation manual (typically 20-30 N·m)

4. **Orient for Right/Left Mounting**:
   - Rotate TK3/TK4 during installation to achieve desired orientation
   - **Advantage of Direct Mount**: More control over final orientation (no adapter rotation during torquing)

### Post-Installation Verification

**Step 1: Leak Check**:
- Pressurize system to operating pressure (slowly if possible)
- Leak-check adapter connections with soap solution or electronic leak detector
- **Critical Points**: Thread joints (adapters Code 1/2/4/6), Rotalock seal (Code 5/8), flange gasket (Code 3/7)

**Step 2: Verify Clearances**:
- Check TK3/TK4 body clearance from compressor housing, piping, adjacent equipment
- Ensure access to DIP switches (TK3+), LCD display (TK4), solenoid coil (all models)

**Step 3: Test Orientation**:
- Verify right/left mounting orientation matches plan (regulator body projects in desired direction)
- Check oil return piping from TK3/TK4 discharge port has adequate slope and no restrictions

**Step 4: Functional Test**:
- Power TK3/TK4 unit (24V AC or DC per model)
- Verify LED indicators, LCD display (TK4), solenoid function per TK3/TK4 commissioning procedure
- Test oil level regulation cycle (see TK3/TK4 operation manual)

### Troubleshooting Installation Issues

**Problem: Adapter Won't Thread Smoothly**

| Cause | Solution |
|-------|----------|
| Thread damage (burrs, cross-threading) | Inspect threads, chase with tap/die if minor damage, replace adapter if severe |
| Wrong adapter type | Verify adapter code matches compressor model in Selection Table |
| Debris in threads | Clean threads thoroughly with wire brush, compressed air |

**Problem: Leak at Thread Joint (Codes 1/2/4/6)**

| Cause | Solution |
|-------|----------|
| Insufficient sealant | Remove adapter, reapply PTFE tape (3-4 wraps) or fresh anaerobic sealant |
| Under-torqued | Increase torque to specification (use calibrated torque wrench) |
| Over-torqued (damaged threads) | Replace adapter and/or repair compressor threads (re-tap or Helicoil insert) |
| Wrong sealant (incompatible with refrigerant) | Use refrigeration-approved sealant (PTFE tape or anaerobic paste) |

**Problem: Leak at Rotalock Connection (Codes 5/8)**

| Cause | Solution |
|-------|----------|
| Damaged O-ring | Replace O-ring with Viton or nitrile (refrigerant-compatible) |
| Under-torqued (not 50 N·m) | Retorque to exactly **50 N·m** with Rotalock service tool |
| Over-torqued (crushed O-ring) | Replace O-ring, retorque to **50 N·m** (no more) |
| Dirty Rotalock seat | Remove adapter, clean Rotalock port seat thoroughly, reinstall with new O-ring |

**Problem: Leak at Flange Joint (Codes 3/7)**

| Cause | Solution |
|-------|----------|
| Gasket damaged or missing | Replace gasket (correct material: nitrile, EPDM, or per compressor spec) |
| Uneven bolt torque | Re-torque bolts in star pattern (even loading critical) |
| Warped flange surface | Inspect flange flatness, resurface if possible, or use thicker gasket/liquid sealant |
| Insufficient bolt torque | Increase torque to compressor manufacturer specification |

**Problem: Insufficient Clearance for TK3/TK4 Body**

| Cause | Solution |
|-------|----------|
| Wrong mounting orientation (right vs left) | Rotate TK3/TK4 on adapter 180° to switch right/left orientation |
| Adapter L1 height too tall | Consider shorter adapter (e.g., Code 7 instead of Code 3 for flange) |
| Compressor location too close to wall/equipment | Relocate compressor or reroute adjacent piping/equipment |

## 6. Frequently Asked Questions

### Q1: How do I know which adapter I need for my compressor?

**Step-by-Step Selection**:

1. **Identify compressor**: Find brand and model on nameplate (e.g., "Bitzer 4G-20.2", "Copeland ZF15K5E")
2. **Use Selection Table** (Section 4): Look up brand → Find model family → Note selection code (0, 1, 2, or 3)
3. **Match to adapter**:
   - Code 0 → No adapter (direct mount)
   - Code 1 → TKX-A001 (UNEF)
   - Code 2 → TKX-A002 (NPT)
   - Code 3 → TKX-A003 (flange 50mm)

**Example**: Bitzer 4G → Selection "0 or 3" → Choose TKX-A003 (flange) OR no adapter (direct mount).

---

### Q2: What does "0 or 3" mean in the Selection Table?

**Two Options Available**:

- **Code 0**: Direct mounting (no adapter) - TK3/TK4 threads directly into compressor
  - **Advantages**: Most compact, lowest cost, fewest leak points
  - **Use when**: Compressor has direct mount provision, space constrained

- **Code 3**: Flange adapter TKX-A003 (50mm spacer)
  - **Advantages**: Flexible mounting, standardized interface, can function as spacer
  - **Use when**: Flange mounting required, need 50mm clearance, or prefer standardized connection

**How to Choose**: Check compressor documentation. If uncertain, **choose Code 3** (flange) for maximum flexibility.

---

### Q3: Can I use the same adapter for right or left mounting?

**Yes, absolutely!** All adapters are **universal for right and left mounting**.

**How It Works**:
- **Right Mounting**: Rotate TK3/TK4 on adapter so regulator body projects to right side of compressor
- **Left Mounting**: Rotate TK3/TK4 on adapter 180° so body projects to left side

**No Special Ordering**: Same adapter part number (e.g., TKX-A001) works for both orientations. Specify "right" or "left" during installation, not during order.

---

### Q4: What is L1 dimension and why does it matter?

**L1 = Additional height** (mm) from mounting face to TK3/TK4 connection point.

**Why It Matters**:
- **Total height**: Base TK3/TK4 height + L1 = Total assembly height
- **Clearance planning**: Must account for L1 when verifying installation space
- **Access**: Higher L1 may improve access to electronic module, but requires more vertical space

**Example**: TK3+ (101mm) + NPT adapter Code 2 (L1 = 34mm) = **135mm total height**

**Planning Tip**: If clearance limited, choose adapter with lower L1 (e.g., Code 7 flange @ 22mm vs Code 3 @ 50mm).

### Q6: What is the difference between TKX-A003 and TKX-A0070?

**Both are 3/4/6-hole flange adapters**, but different heights:

| Feature | TKX-A003 (Code 3) | TKX-A0070 (Code 7) |

| **L1 Height** | 50 mm | 22 mm |

| **Use Case** | Standard clearance, general purpose | Low-profile, space-constrained |
| **As Spacer** | 50mm spacer function | 22mm spacer function |

**Choose Code 3**: Standard applications, need 50mm clearance/spacer  
**Choose Code 7**: Limited vertical space, prefer lower profile

### Q7: Which Rotalock adapter do I need?

**Two Rotalock sizes available**:

| Code | Size | Port Diameter | L1 Height | Application |
|------|------|---------------|-----------|-------------|
| **5** (TKX-A005) | 1" ¾ - 12 UNF | 1.75" (44.5mm) | 53.4 mm | Smaller Rotalock ports |
| **8** (TKX-A008) | 2" ¼ - 12 UNF | 2.25" (57.2mm) | 41.4 mm | Larger Rotalock ports |

**How to Determine**:
1. **Measure port diameter** with caliper: 1.75" → Code 5; 2.25" → Code 8
2. **Check equipment nameplate** or technical documentation (often specifies Rotalock size)
3. **Inspect existing Rotalock connection** (if replacing component)

### Q14: How do I install thread adapters properly?

**Quick Guide**:

1. **Clean threads** on compressor mounting port (remove debris, old sealant)
2. **Apply sealant**: PTFE tape (2-3 wraps) OR anaerobic thread sealant
3. **Thread adapter** hand-tight into compressor port
4. **Orient adapter** for desired right/left mounting (account for ~30-45° rotation during torquing)
5. **Torque to specification** with calibrated torque wrench:
   - UNEF (Code 1): 25-35 N·m
   - NPT (Code 2): 30-40 N·m
   - GAS (Code 4): 30-40 N·m
   - M36 (Code 6): 35-45 N·m
6. **Install TK3/TK4** onto adapter, orient regulator body for right/left as planned

**Full Details**: See Section 5 (Installation Guidelines).
